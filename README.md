# Aule

## Project Structure

```
packages/       Rust workspace crates
app/            Front-end application (Bun/TypeScript)
docs/           Documentation
```

## Prerequisites

- [Rust](https://rustup.rs/) (stable)
- [Bun](https://bun.sh/)

## Getting Started

### Rust

```sh
cargo build
cargo test
```

### Front-end

```sh
cd app
bun install
bun run index.ts
```
